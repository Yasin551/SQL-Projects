# Bank Transaction System using SQL Triggers 🏦

This project demonstrates a **Bank Transaction System** where all deposits and withdrawals are automatically tracked using **SQL triggers**.  

---

## Features ✅
- Tracks **deposits** and **withdrawals** automatically.
- Maintains a **transaction log** with old and new balances.
- Easy to extend for **transfers between accounts**.
- Ideal for learning **SQL triggers** and **transaction management**.
